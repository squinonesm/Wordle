# Wordle para Android

Un juego estilo Wordle desarrollado desde cero en Android Studio utilizando Java. El proyecto es una recreación del popular juego de palabras donde los jugadores deben adivinar una palabra secreta en un número limitado de intentos.

## Características
- Juego de palabras con interfaz intuitiva.
- Funcionalidad básica de adivinar palabras.
- Feedback visual para indicar letras correctas y posiciones.

## Estado del Proyecto
Actualmente, el proyecto está en desarrollo activo. Se planean futuras mejoras, como:
- Implementación de nuevas características.
- Optimización del diseño de la interfaz.

## Tecnologías Utilizadas
- **Lenguaje:** Java
- **IDE:** Android Studio
